# Superstore Sales Data Analytics Project

## Overview

This project was completed as part of the ApexPlanet Software Pvt. Ltd. Data Analytics Internship Program.

The objective of this project was to analyze the Superstore Sales Dataset and derive meaningful business insights through data cleaning, exploratory data analysis, SQL querying, dashboard development, statistical analysis, customer segmentation, predictive modeling, and automation.

---

## Dataset

**Dataset Name:** Superstore Sales Dataset

The dataset contains retail transaction information including:

- Orders
- Customers
- Products
- Categories
- Sales
- Profit
- Regions
- Shipping Details

---

## Key Findings

1. Technology category generated the highest sales.
2. West region achieved the strongest business performance.
3. High sales products were not always highly profitable.
4. Customer spending was concentrated among a smaller group of customers.
5. Seasonal sales trends were identified across the dataset.

---

## Business Recommendations

- Increase focus on high-performing product categories.
- Improve profitability of low-margin products.
- Strengthen customer retention strategies.
- Use seasonal trends for inventory planning.
- Expand successful regional strategies.
